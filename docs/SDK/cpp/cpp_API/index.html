接口说明

本章列出各版本 xCore SDK 所支持的接口和功能简述。不同开发语言的版本对接口的功能定义基本一致，但是参数、返回值和调用方法会有区别。

C++：实例化 rokae::Robot 类​

根据机器人构型和轴数不同，C+＋ 版本的 SDK 提供了下列几个可供实例化的 Robot 类，初始化时会检查所选构型和轴数是否和连接的机器人匹配：

类名适用机型
xMateRobot协作6 轴
xMateErProRobot协作7 轴
StandardRobot工业6 轴
xMateCr5Robot协作CR5 轴
PCB4Robot工业4 轴
PCB3Robot工业3 轴

id -指令 ID ，用于执行信息反馈
